## Дизайн-концепция

### Общий стиль
- **Тема:** Тёмная, техно-минимализм с акцентами на тему сигналов и волн
- **Вдохновение:** Осциллограммы, спектры сигналов, constellation diagrams QAM
- **Цветовая палитра:**
  - Фон: `#0a0a0f` (почти чёрный с синим оттенком)
  - Основной акцент: `#00d4ff` (cyan / сигнальный голубой)
  - Вторичный акцент: `#7c3aed` (фиолетовый / AI-тема)
  - Текст: `#e2e8f0` (светло-серый)
  - Muted: `#64748b` (приглушённый серый)
- **Шрифты:**
  - Заголовки: `JetBrains Mono` (инженерный vibe)
  - Текст: `Manrope`
- **Фоновые элементы:** Анимированные синусоиды, constellation diagram QAM как декоративные элементы, частицы / сетка

### Анимации
- Hero: текст печатается (typewriter), фоновая волна пульсирует
- Секции: появляются при скролле (fade-in + slide-up) через framer-motion
- Карточки проектов: hover — glow + подъём + tilt
- Timeline: линия рисуется при скролле
- Навигация: backdrop-blur при скролле
- Курсор: мягкое свечение, следующее за мышью

## Правила для Claude Code

1. **Всегда используй TypeScript** — никаких `.js` файлов
2. **Tailwind CSS** — основной инструмент стилизации
3. **Framer Motion** — для всех анимаций
4. **Компоненты — функциональные** с `"use client"` где нужны хуки
5. **Данные** — бери из этого CLAUDE.md файла (проекты, навыки, текст)
6. **Цвета** — строго по палитре из дизайн-концепции
7. **Шрифты** — JetBrains Mono для заголовков, Manrope для текста
8. **Адаптивность** — mobile-first, breakpoints: sm/md/lg/xl
9. **Не используй стоковые изображения** — только градиенты, SVG, геометрия
10. **Каждый компонент** — отдельный файл в `src/app/components/`
